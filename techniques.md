<p style="font-size:12pt">Open Restoration is an archive of ecological restoration manuals. This collection includes any document that is publicly available and provides instruction on how to restore ecological integrity to an area. If there are any additions, please <A href="mailto:tim.a@uwaterloo.ca">email me</a>!</p>
